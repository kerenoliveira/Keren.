01

Introdução

A Keren. — representada por Keren Layane de Oliveira Silva, profissional autônoma de estratégia, design e criação digital, com sede em Rio Verde – GO — tem o compromisso de proteger a privacidade de todas as pessoas que visitam este site ou entram em contato para contratar serviços.

Esta Política de Privacidade explica de forma clara e transparente quais dados pessoais são coletados, por que são coletados, como são utilizados e quais são os seus direitos como titular, em conformidade com a Lei Geral de Proteção de Dados Pessoais (LGPD — Lei nº 13.709/2018).

Ao utilizar este site ou enviar qualquer informação de contato, você concorda com os termos descritos nesta política.

Resumo direto: Seus dados são usados apenas para que eu possa responder seu contato, elaborar propostas e executar projetos contratados. Não vendemos, não compartilhamos comercialmente e não fazemos spam.

02

Dados coletados

Os dados coletados variam de acordo com a forma como você interage com este site:

Via formulário de contato
  • Nome completo — para identificação e personalização do atendimento
  • E-mail — canal principal de comunicação
  • Nome da empresa ou negócio — para entendimento do contexto
  • Tipo de projeto — para direcionamento da proposta
  • Mensagem livre — descrição das necessidades do projeto
Automaticamente, pela navegação
  • Endereço IP — registrado automaticamente pelo servidor
  • Tipo de dispositivo e navegador — para garantir a compatibilidade do site
  • Páginas acessadas e tempo de navegação — por meio de ferramentas de análise
  • Localização aproximada (cidade/estado) — com base no IP, sem precisão de endereço
Em projetos contratados
  • Documentos de identificação — CPF ou CNPJ, para formalização de contratos
  • Informações do negócio — fornecidas no briefing de projeto
  • Materiais enviados — fotos, logos, textos e outros arquivos do cliente
03

Como usamos seus dados

Seus dados são utilizados exclusivamente para as finalidades abaixo, com base nas hipóteses legais da LGPD:

  • Responder ao contato inicial — retorno dentro de 48 horas úteis, conforme prometido no site
  • Elaborar e enviar propostas comerciais — com base nas informações fornecidas no formulário
  • Executar projetos contratados — utilizando os dados e materiais entregues durante o processo
  • Formalização contratual — geração de contratos e notas fiscais quando aplicável
  • Comunicação sobre o andamento do projeto — atualizações, revisões e entrega final
  • Melhorias no site — análise de navegação para otimizar a experiência do usuário

Seus dados nunca são utilizados para envio de newsletters sem autorização, publicidade de terceiros ou qualquer finalidade diferente das listadas acima.

04

Cookies e rastreamento

Este site pode utilizar cookies — pequenos arquivos de texto armazenados no seu navegador — para garantir o funcionamento adequado e coletar dados de navegação de forma anônima.

Tipo Finalidade Duração Categoria
Técnicos Funcionamento básico do site Sessão Essencial
Análise (Google Analytics) Estatísticas de acesso e comportamento de navegação de forma anônima Até 2 anos Analítico

Você pode desativar os cookies diretamente nas configurações do seu navegador. A desativação dos cookies analíticos não prejudica a utilização do site — apenas a coleta de dados de navegação é interrompida.

Este site é hospedado no GitHub Pages, que pode coletar dados técnicos de acesso de acordo com os próprios termos de uso do GitHub. Consulte a política de privacidade do GitHub para mais detalhes.

05

Compartilhamento de dados

Seus dados pessoais não são vendidos, alugados ou cedidos a terceiros para fins comerciais.

Em casos específicos, dados podem ser compartilhados com:

  • Ferramentas de trabalho — como Google Workspace (Gmail), utilizadas exclusivamente para comunicação e organização interna de projetos, com proteção de dados adequada
  • Prestadores de serviço técnico — plataformas como GitHub Pages (hospedagem do site), que possuem suas próprias políticas de privacidade
  • Autoridades competentes — quando exigido por lei, ordem judicial ou obrigação regulatória

Nenhum dado é compartilhado com parceiros comerciais, agências de publicidade ou qualquer outra empresa para fins de marketing ou prospecção.

06

Armazenamento e retenção

Seus dados são armazenados de forma segura e são mantidos pelo tempo necessário para cumprir as finalidades descritas nesta política.

  • Contatos sem conversão em projeto — mantidos por até 12 meses para eventual retomada de contato, depois disso são eliminados
  • Dados de projetos realizados — mantidos por até 5 anos para fins de documentação, suporte e cumprimento de obrigações legais e fiscais
  • Dados de navegação anônimos — mantidos por até 2 anos pelas ferramentas de análise

Após os prazos indicados, os dados são eliminados de forma segura ou anonimizados, de modo que não seja mais possível identificar o titular.

07

Seus direitos como titular

A LGPD garante a você, como titular dos dados, os seguintes direitos — que podem ser exercidos a qualquer momento mediante solicitação:

  • Confirmação e acesso — saber se seus dados são tratados e ter acesso a eles
  • Correção — solicitar a atualização de dados incompletos, inexatos ou desatualizados
  • Anonimização, bloqueio ou eliminação — de dados desnecessários ou tratados em desconformidade com a lei
  • Portabilidade — receber seus dados em formato estruturado para uso em outro serviço
  • Eliminação — solicitar a exclusão dos dados tratados com base em seu consentimento
  • Revogação do consentimento — retirar a autorização para o tratamento de dados a qualquer momento
  • Oposição — contestar o tratamento realizado em desacordo com a lei
  • Informação — ser informado sobre as entidades públicas e privadas com as quais dados foram compartilhados

Para exercer qualquer um desses direitos, envie um e-mail para admkerenoliveira@gmail.com com o assunto "Direitos LGPD". O retorno será feito em até 15 dias úteis.

08

Segurança dos dados

São adotadas medidas técnicas e organizacionais para proteger seus dados contra acesso não autorizado, perda, alteração ou divulgação indevida:

  • Comunicação por canais com criptografia (Gmail com TLS, WhatsApp com criptografia ponta a ponta)
  • Armazenamento de arquivos de projeto em plataformas com controle de acesso
  • Contratos e documentos assinados digitalmente ou mantidos de forma segura
  • Acesso restrito aos dados — apenas a responsável pelo projeto tem acesso às informações do cliente

Em caso de incidente de segurança que possa causar dano relevante, os titatulares afetados serão notificados dentro do prazo legal.

09

Menores de idade

Os serviços da Keren. são destinados a pessoas maiores de 18 anos ou empresas. Este site não coleta intencionalmente dados de menores de 18 anos.

Caso um responsável legal identifique que dados de um menor foram fornecidos sem autorização, solicitamos contato imediato para que tais informações sejam prontamente eliminadas.

10

Alterações desta política

Esta Política de Privacidade pode ser atualizada periodicamente para refletir mudanças na legislação, nos serviços prestados ou nas práticas adotadas.

Alterações relevantes serão comunicadas com destaque no site. A data de "última atualização" no topo desta página indica quando ocorreu a revisão mais recente.

O uso continuado do site após a publicação de alterações implica a aceitação dos novos termos.
